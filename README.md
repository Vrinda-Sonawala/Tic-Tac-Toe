# Tic Tac Toe Game

A simple Tic Tac Toe web game built using HTML, CSS, and JavaScript.

## Technologies Used
HTML, CSS, JavaScript

## Features
- Two-player game (X and O)
- Winner and draw detection
- Reset and New Game option
- Interactive and responsive UI

## Description
Players take turns clicking on the 3Ã—3 grid.  
The game checks winning patterns after each move and displays the result.  
If all moves are completed without a winner, the game ends in a draw.
